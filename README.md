## Perl Mojolicious lite app that reports  Covid-19 cases in Romania by county.
*You can filter the data and download it in xlsx format*
### Backend:
* Perl
* Mojolicious::Lite
* Dbix::Class
* Minion
* Mojo::AssetPack
* MySql

### Frontend:
* Vue.js
* Bootstrap
* axios
